
### 方法2：Docker部署
```bash
# 使用Docker Compose启动所有服务
docker-compose up -d

# 查看服务状态
docker-compose ps
```

### 方法3：手动部署
```bash
# 启动后端API
cd backend
npm install
npm start

# 启动前端（新终端）
cd frontend
npm install
npm run build
serve -s build
```

## 🔧 技术架构

### 后端
- **框架**: Node.js + Express
- **数据库**: SQLite（轻量级，无需额外配置）
- **认证**: JWT Token
- **API**: RESTful设计

### 前端
- **框架**: React + TypeScript
- **UI组件**: Ant Design
- **图表**: Recharts
- **状态管理**: React Hooks + Context

### 部署
- **容器化**: Docker + Docker Compose
- **Web服务器**: Nginx（生产环境）
- **监控**: 健康检查和日志记录
